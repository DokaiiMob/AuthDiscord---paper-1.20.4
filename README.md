# AuthDiscord

**AuthDiscord** — плагин авторизации для Minecraft серверов, интегрированный с Discord. Позволяет игрокам подтвердить свою личность через Discord бота перед получением доступа к игре, обеспечивая дополнительный уровень безопасности.

## Основные возможности

- Авторизация через Discord с уникальными временными кодами  
- Проверка участия игрока в указанном Discord сервере  
- Блокировка игровых действий для неавторизованных пользователей  
- Отображение Discord тегов в игре  
- Хранение данных привязок в SQLite базе  
- Административные команды для управления привязками и конфигурацией


## Как это работает
1. Игрок заходит на сервер и получает уникальный код авторизации.  
2. Отправляет код в личное сообщение Discord боту.  
3. Бот проверяет, что игрок состоит в вашем Discord сервере.  
4. При успешной проверке игрок получает доступ к полноценной игре.


## Требования
- Minecraft сервер Paper/Spigot 1.20 или выше  
- Java 21 или новее  
- Discord бот с токеном и необходимыми разрешениями  


## Установка
### 1. Создание Discord бота
- Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)  
- Создайте новое приложение и добавьте бота  
- Скопируйте токен бота  
- Включите в настройках бота Intent "Message Content Intent"  

### 2. Установка на сервер
* Поместите файл `authdiscord-1.0.0.jar` в папку `plugins/` вашего сервера
* Перезапустите сервер для создания конфигурационных файлов

### 3. Настройка
Отредактируйте файл `plugins/AuthDiscord/config.yml`:

```yaml
bot_token: "ВАШ_ТОКЕН_БОТА"
guild_id: "ID_ВАШЕГО_ДИСКОРД_СЕРВЕРА"
code_length: 8
code_timeout: 5
```

### 4. Приглашение бота на сервер
Используйте ссылку, заменив `ID_БОТА` на Client ID вашего бота:
https://discord.com/oauth2/authorize?client_id=ID_БОТА&scope=bot&permissions=67584


### 5. Перезапуск сервера
Перезапустите сервер для применения настроек.

## Команды
* `/unlink <игрок>` — удалить привязку игрока (требуются права `authdiscord.admin`)
* `/reloadauth` — перезагрузить конфигурацию плагина (требуются права `authdiscord.admin`)

## Права доступа
* `authdiscord.admin` — права администратора плагина (по умолчанию для OP)
